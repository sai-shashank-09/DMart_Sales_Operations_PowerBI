# DMart Sales & Operational Intelligence Dashboard | Power BI

## Problem Statement
DMart aims to improve business decision-making using data-driven insights. This project analyzes historical retail transaction data to understand sales performance, customer behavior, pricing patterns, and operational efficiency through an interactive Power BI dashboard.

## Objectives
- Analyze monthly, quarterly, and yearly sales trends
- Understand customer demographics and purchase behavior
- Evaluate product performance and discount impact
- Monitor operational metrics such as shipping, payments, and order status

## Dataset Overview
- 25,000+ retail transactions
- Customer demographics (Age, Gender)
- Product details (Category, Brand Type, Pricing, Discounts)
- Order and delivery timeline
- Website engagement metrics (Clicks, Time Spent)
- Operational data (Payment Mode, Order Status, Shipping Mode)

## Tools & Technologies
- Power BI
- DAX (Calculated Columns and Measures)
- Excel / CSV
- Data Visualization and Business Intelligence

## Key Insights
- Identified peak and low sales periods
- Analyzed revenue contribution by product types
- Studied impact of discounts and website engagement on purchases
- Highlighted shipping efficiency and order cancellation patterns
